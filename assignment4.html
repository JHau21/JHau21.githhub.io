Challenge 5: Accessibility Concerns

At the moment I think there are a variety of accessibility concerns with Challenge 4. The largest conern, in my opinion is addressing the best practice of making everything on the website accessible via keyboard. Since Challenge 4 contains so many different links and is already nested somewhat far down the page, a user who is unable to use a mouse or scroll pad properly will have quite a bit of trouble getting to a desired link, assuming that link is pretty far down. Said user would need to tab through essentially all of the content on the page to get to the lowest link in the challenge. In my opinion, the best way to remedy this would be to add a table of contents at the top to allow the user to quickly access challenge 4. This would also be good to generally access every challenge too because it would keep users from needing to tab or scroll through the entire page.
