Challenge 3: Create a Form

Part 3:

For this part of the assignment I decided to use the spellcheck attribute in my input tags to check the user spellings as they enter characters. To see where I found this attribute click HTML Attributes and scroll down to the spellcheck tag. This attribute can be used in input elements or related fields, like textarea, and will check the user spelling as they complete words. It functions essentially like the spell check in any text editor and red underlines misspelled or not understood words
